About "Jobs-App"

Overall

This page explains the website features.
The website connect between employers and employees, help them to find a Job or find employee.
All the users can Like / Unlike other users, can message to each other at live chat with signalR
and can edit his own details, even upload photos.

Find Employee

User can find other users by search field or by filters as age, employee / employer, profession, last active ,newest member or reset filters.
The result will be as cards that include all the important details of the user.
The user can navigate to user details by hover the card and click the user icon (left), also the user can like / unlike the user by click the heart icon (middle),
and can message to user by click the message icon (right).

Edit profile

The member can edit his details, general details, resume and upload photos (until the admin improve the photo) and set main photo as well.

List

At list page the user can see member that i like them or member who like me.
If the user dont like any more other user, he can unlike and the member will be disappear.

Messages

At messages page the user can see messages unread, inbox or outbox from all the members.
The user can manage the messages by delete the messages.

Footer

The member can contact with the author of the website.

Error

This page for admin use only, check the error that we can get during use the web application.

Admin

When the admin login to the web, he will see the Admin page.
At this page he can manage the roles of other members, if them Member, Moderator or even Admin.
Also when member add photo he need to improve the photo. After the admin improve the photo the member will see the photo.
